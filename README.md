# 📘 Taller 2: Python — Estructuras de Control y Funciones

Este repositorio contiene un **taller práctico** de Python enfocado en:
- Manejo de variables y entrada de datos
- Condicionales
- Ciclos `for` y `while`
- Listas y diccionarios
- Creación y uso de funciones [1](https://github.com/carloscastrox/t2_3406211_py)  

---

## 🎯 Objetivo
Fortalecer la lógica de programación mediante la implementación de **ejercicios progresivos** organizados por secciones temáticas. [1](https://github.com/carloscastrox/t2_3406211_py)  

---

## ✅ Requisitos
- Python 3.x (recomendado)  
- Editor de código (VS Code u otro)  

> *(Opcional)* Puedes usar un entorno virtual si deseas mantener tu instalación ordenada.
